Modal Form Example

This is an example of a modal email-form. It uses javascript for a quick client-side validation and php for server-side validation.
There is visual feedback and a simple antispam check. The form returns a success message when is sent and does not need page refresh.
It resets itself everytime the user sents a message or closes the modal.
